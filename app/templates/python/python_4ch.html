基本数据类型   

我们按一定的数据结构存储数据时，在“数据结构”之下，还有一项更基础的关于数据的属性，那就是“数据类型”。

对于Python来说，最基本的、最重要的有4种数据类型：

integer(整数,略写为int），float(浮点，小数）， string（字符串，略写为str）， boolean(布尔值，略写为bool）。

有人可能会疑惑为什么需要把int和float分开来，而不是弄一个number(数字）类呢？ 这和数字在计算机内的表示和储存方式有关。对于初学者来说，不必细究，记住 int 和 float 的分类就行。感兴趣的同学，可以自行搜索。

接下来简要介绍一下这几类数据类型：

int（整数类）

整数就是诸如 0,1,2……9999等这类的整数，和数学上的整数的定义一样，int类可以直接实现数学上的加减乘除、 乘方、开方、大小比较等一系列的数学运算，（Python有Math包可以简单地进行数学计算，后面会提到）。

在Console中，输入 type('variable')可以查看该变量的数据类型。如：

>>> type(1)
<class 'int'>

class 后面的，就是该变量（例子中是1）的数据类型，可以看到， 1的类型是int(整数类)。

int类最基础、也是最重要的使用方法是作为数学上的计算。在python语言中，一般的数学运算符号及其使用方法如下：

序号 符号 含义 例子
1 + 加法 1+1=2
2 - 减法 2-1=1
3 * 乘法 2*3=6
4 / 除法 8/2=4
5 ** 乘方（开方） 2**3=8；9**0.5=3
4 % 除法取余 8%3=2
4 // 除法取整 10//3=3

我们在进行一般的数学运算时，常常会用到括号。在Python语言中，无法直接使用数学意义上的中括号和大括号（这两个括号另有含义），所有数学运算上的括号都由小括号表示，不同的运算顺序用小括号的嵌套层级来决定，计算时由内层计算到外层。来看几个例子。

>>> ((1 + 2) * ((3 + 4)/5))
4.199999999999999

计算顺序是先计算(3+4)，再计算 7/5 和 （1 + 2），最后是二者结果的乘积。

除了基本的数学运算，int类通常还会在很多数据结构中担任指针（索引）的作用，这个我们在具体介绍到常用数据结构的时候再详解释。

float（浮点数）

在使用时，浮点数就是数学意义上的小数，用浮点数这个名字是因为，在科学计数法法中，小数的小数点是可“浮动”的， 1.2*10^3 与 12*10^2是等价的。浮点数在python内的表达，既可以使用一般的小数记法如 1.23， 2.333，16.00 ，也可以用科学计数法法。但请记得在python语言中科学计数法写作 1.23e3,表示 1.23 * 10^3.

float类的使用方法基本上也是数学意义上的，和int类类似。值得注意的是，float无法担任更高级的数据结构的指针。

str（字符串）

字符串是所有文字的集合，包括我们平时使用的各种自然语言。通常用引号包围起来，双引号单引号均可，但切记引号要成对出现。比如‘integer’, "integer"都是是合法的， 但是 'integer"则无法被正确的识别。当然，引号可以层级包裹。如果你在字符串中想用引号表示引用，那么记住在外层和内层使用不同的引号。

如："he is a 'good' man."是合法的， 'he is a "good" man.' 也是合法的。

在这里需要额外注意的是，用引号引起来的数字，也是字符串。 '1', '1.23'这样的数据，其类型依然是字符串，而非int或者float。

如果想将字符串的'1', '1.23'转为数字，可以使用 int(), float()方法。

>>> b = '1.23'
>>> type(b)
<class 'str'>
>>> b = float(b)
>>> b
1.23
>>> type(b)
<class 'float'>

对于字符串来说，它的长度是它的重要属性。字符串的长度是指一串字符串所含的“字符（还包括空格和符号）”的总长度，举例来说：'hello world!' 的长度是12， '人生苦短，我用python'的长度是13。

如果想获得一串字符串的长度，可以使用len()方法。

>>> len('hello world!')
12
>>> len('人生苦短，我用python')
13

有的时候，我们可能想要将两个字符串拼在一起，在python中，你可以简单地使用“+”运算来实现这一效果。

>>> a = 'hello '
>>> b = 'world!'
>>> a+b
'hello world!'

那么，拆解字符串是否可以使用“-”运算呢？当然不能！

如果你想将一个字符串分为几个字符串，你可以使用python内置的 split()方法，使用这个方法，你需要对你的字符串设定一个拆分的片段。话不多说，我们通过例子来讲解。

>>> s = "i'm a student"
>>> s.split(' ')
["i'm", 'a', 'student']

>>> s2 = 'a1b1c1d'
>>> s2.split('1')
['a', 'b', 'c', 'd']

第一个例子中，原字符串被分成了三个部分["i'm", 'a', 'student']（这里储存这三个字符串的数据结构叫做数组，我们下一章会讲到），而我们使用的命令 s.split(' ')，注意两个引号中间有一个空格。看到这里你可能已经明白了，我们使用了空格符作为拆分的片段来拆分了原字符串，通过观察原字符串我们发现这三个子字符串之间确实是由空格隔开的。

第二个例子可能更明显一些，我们使用了字符串的数字：‘1’来作为拆分的字段，结果也很明显。

除了拼凑与拆分，对于英文字符串，还有一个很基本的操作：切换大小写。

如果你想要将一个字符串首字母大写，你可以使用 str.capitalize() 方法。

>>> a = 'hello world!'
>>> a.capitalize()
'Hello world!'

将字符串完全大写，可以使用 str.upper() 方法。

>>> b = 'hello. nice to meet you, too. this is a sample sentence.'
>>> b.upper()
'HELLO. NICE TO MEET YOU, TOO. THIS IS A SAMPLE SENTENCE.'

将字符串完全小写，可以使用 str.lower() 方法。

>>> b = 'THIS IS AN UPPER SENTENCE.'
>>> b.lower()
'this is an upper sentence.'

除了这些常用操作外，str类还有一种常用操作，叫做“切片（slice)”，可以理解为提取字符串中的某一片段，这个我们在后面介绍线性表的时候还会提到，现在先不详述。

boolean（布尔值）

这是一对相对的值，True和False。如字面意思，这是表示真与假的逻辑值。

在Python中，boolean值既可以直接用True\False来表示，也可以通过运算来表示。 比如 1>2 （注意数字没有被引号括起来，是int）等价于False， 2 > 1 等价于 True. 在进行元素的比较时，“==”表示 “相等”， “!=”表示“不等”， “>=”表示“大于等于”， “<=”表示小于等于。我们来看几个元素比较和布尔值的例子。

>>> True == True
True
>>> False == False
True
>>> True != False
True
>>> True != True
False
>>> 2 >= 2
True
>>> 'apple' >= 'boss' #字母的大小关系根据字母表顺序定的，位置靠前的 < 位置靠后的
False
>>> 'apple'<'boss'
True
>>> type(233) == int
True
>>> type('233') == int
False
>>> (2 > 1) ==True
True
>>> type(2 > 1) == bool
True

相信大家可以猜到，这一对表示真与假的逻辑值主要被用来做条件判断从而进行逻辑控制，这个我们在讲到基本控制流的时候会详细提及。

那么，以上就是关于Python的四大基本数据类型的说明，下面，我们就来学习如何把这些不同类型的数据组织起来吧！下一章，基本数据结构！
