4.2 逻辑类型   2017.10.11

逻辑类型的数据可以被看作是一类只有真和假两种状态的数据，就如同在二进制中常用1表示真，0表示假一样。在输入逻辑类型的数据时，MATLAB将所有非0数值当作真，将0当作假。在输出逻辑类型的数据时，MATLAB将真输出为1，将假输出为0。生成逻辑类型的数据可以使用logical函数，其用法与生成数值类型的函数们一样。

逻辑类型的数据进行运算时需要使用关系操作符和逻辑运算符。MATLAB中的关系操作符有小于（<）、小于或等于（<=）、大于（>）、大于或等于（>=）、等于（==）、不等于（～=）六种，逻辑运算符有与（&）、或（|）、非（～）三种。

在脚本编辑器或命令行输入：

a = 1; b = 2; c = 1;
a == b, (a ~= b) & (a == c)

输出结果为：

ans =
  logical
   0
ans =
  logical
   1
