4.1 数值类型   2017.10.08

数值类型数据的分类

在MATLAB中，基本的数值类型有整数、单精度浮点数和双精度浮点数3类，如下表所示。

数据类型 数据格式
整数 int8, uint8,
int16, uint16,
int32, uint32,
int64, uint64
单精度浮点数 single
双精度浮点数 double

在未加说明与特殊定义时，MATLAB对所有数值按照双精度浮点数类型进行存储和操作。在需要时，可以使用整数类型或单精度浮点数类型进行运算和操作。整数类型和单精度浮点数类型的优点在于占用内存小，但首先要在能够满足精度要求的前提下。

整数类型

MATLAB提供了8种整数类型。这8种整数类型占用的位数不同，因此能够表示的数值范围也不同，如下表所示。

整数类型 数值范围 转换函数
有符号8位整数 -27～27-1 int8
无符号8位整数 0～28-1 uint8
有符号16位整数 -215～215-1 int16
无符号16位整数 0～216-1 uint16
有符号32位整数 -231～231-1 int32
无符号32位整数 0～232-1 uint32
有符号64位整数 -263～263-1 int64
无符号64位整数 0～264-1 uint64

转换函数中，数字代表该类型所占用的位数，u代表unsigned（无符号）。使用转换函数，可以将浮点型数值转换为整数数值或在整数数值类型之间相互转换。在将浮点型数值转换为整数数值的过程中，MATLAB默认将待转换数值转换为与之最为接近的整数值，若小数部分位0.5，则转换后的结果为与该浮点数最接近的两个整数中绝对值较大的一个。在整数数值类型之间转换的过程中，待转换的数值不应超过待转换的整数类型能够表示的范围，否则就会出现错误，运算结果为该整数类型能够表示的最大值或最小值。

在脚本编辑器或命令行输入：

int8(2.1)

输出结果为：

ans =
  int8
   2

在脚本编辑器或命令行输入：

int8(2.5)

输出结果为：

ans =
  int8
   3

在脚本编辑器或命令行输入：

int8(1024) %int8类型能够表示的最大值为127

输出结果为：

ans =
  int8
   127

在脚本编辑器或命令行输入：

int16(1024) %int16类型能够表示的最大值为32767

输出结果为：

ans =
  int16
   1024

MATLAB还包含了几类取整函数，也可以把浮点数转换成整数。这些取整函数及相应的转换方式如下表所示。

函数 运算法则 示例
floor(x) 向下取整 floor(1.2)=1
floor(2.5)=2
floor(-2.5)=-3
cell(x) 向上取整 ceil(1.2)=2
ceil(2.5)=3
ceil(-2.5)=-2
round(x) 取最接近的整数
如果小数部分是0.5，则向绝对值大的方向取整 		round(1.2)=1
round(2.5)=3
round(-2.5)=-3
fix(x) 向0取整 fix(1.2)=1
fix(2.5)=2
fix(-2.5)=-2

浮点数类型

MATLAB中提供了单精度浮点数类型和双精度浮点数类型。单精度浮点数能够表示的数值范围和数值的精度都比双精度浮点数小，更加具体的差别超出了本教程的难度范围，不做介绍。

双精度浮点数参与运算时，返回值的类型依赖于参与运算的其他数据类型。参与运算的其他数据为逻辑型、字符型时，返回结果为双精度浮点型；参与运算的其他数据为整数型时，返回结果为相应的整数类型；参与运算的其他数据为单精度浮点型时，返回结果为相应的单精度浮点型。

在MATLAB中，单精度浮点型不能与整数类型进行算术运算。

在脚本编辑器或命令行输入：

a = uint32(120); b = single(22.809); c = 73.226;
ab = a * b %整数与单精度浮点型进行运算

输出结果为：

错误使用  * 
整数只能与相同类的整数或标量双精度值组合使用。

在脚本编辑器或命令行输入：

ac = a * c %120*73.226=8787.12

输出结果为：

ac =
  uint32
   8787

在脚本编辑器或命令行输入：

bc = b * c

输出结果为：

bc =
  single
  1.6702e+03

在脚本编辑器或命令行输入：

str = 'hello'

输出结果为：

str =
hello

在脚本编辑器或命令行输入：

newstr = str - 44.3

输出结果为：

newstr =
   59.7000   56.7000   63.7000   63.7000   66.7000

在脚本编辑器或命令行输入：

whos

输出结果为：

  Name        Size            Bytes  Class     Attributes

  a           1x1                 4  uint32
  ac          1x1                 4  uint32
  b           1x1                 4  single
  bc          1x1                 4  single
  c           1x1                 8  double
  newstr      1x5                40  double
  str         1x5                10  char

在MATLAB中，whos函数可以查看工作区中变量的属性。如果不指明对象，则返回工作区中所有变量的属性，如果指明对象，则返回所指明对象的属性。如在脚本编辑器或命令行输入：

whos a ac

输出结果为：

  Name        Size            Bytes  Class     Attributes

  a           1x1                 4  uint32
  ac          1x1                 4  uint32

复数

MATLAB中默认使用字符i或j作为虚部标志。创建复数时，可以直接按照复数形式输入或者使用complex函数。MATLAB中有关复数的相关函数如下表所示。

函数 说明
real(z) 返回复数z的实部
imag(z) 返回复数z的虚部
abs(z) 返回复数z的模
angle(z) 返回复数z的辐角
comj(z) 返回复数z的共轭复数
complex(a, b) 以a为实部，b为虚部创建复数

无穷量和非数值量

MATLAB中使用inf（Inf）和-inf（-Inf）分别代表正无穷量和负无穷量，nan（NaN）表示非数值量。关于无穷量和非数值量的讨论超出本教程的范围，因此只对二者做最基本的介绍。
