什么是Python   简单叙述      

（本教程是完全面向新手的Python3教程，就算你之前完全没有接触过编程，也丝毫不必担心）

简单来说，Python是一种编程语言。更具体的，Python是一种面向对象的解释性编程语言，看到这，你可能会对“面向对象”和“解释性”感到疑惑，但不用担心，我们会在后面的教程中谈到，现阶段不用理解这两个概念。

作为编程语言，Python能做的事情很多，它可以被用于开发桌面软件、网站（本网站的后台使用的就是Python）、游戏，也可以用来收集网络数据、进行数据处理、进行机器学习等等。不过，如果想要实现同样的全部或部分功能，其他的编程语言，诸如性能极佳的C++, 功能全面的Java， 网站后台领域运用广泛的php，科学计算领域鼎鼎有名的matlab、R语言等也能胜任，那么我们为什么要选择学习python呢？

这是因为Python这门语言具有以下特点：

  1. 1.语言精炼，符号简洁。实现同样的功能，python所需要的代码量要比其他的绝大部分语言要少，同时，相比其他语言使用各种符号(如大括号，分号）来实现代码分块等功能，python使用了缩进的方式，所以python代码可阅读性会更强一些，这意味着我们在学习、阅读代码时的体验会更好。当然，在维持低代码量的同时，python也保持了很强的逻辑性。
  2. 2.扩展丰富、强大。Python所为大家津津乐道的一个地方在于，它拥有众多业内高级工程师编写、维护的扩展（packages），或称之为“包”。这些包使得语言的使用者可以在不花费时间精力处理底层逻辑的情况下，通过简单地对扩展（package）的引用实现一些高级功能。
  3. 3.业界运用广泛。Python这门语言现在已经得到了业内的广泛使用，许多大型网站，如Ins, Quora, 知乎，豆瓣等的后台就是由Python编写的，同时大量的公司内部的网络数据收集（主要是爬虫）使用的是python语言，在科学计算领域和近年兴起的机器学习领域（tensorflow），python也非常的活跃。 所以，你大可不必担心学习这门语言之后找不到实际的用途。
  4. 4.可互动性。Python支持用命令行交互的方式运行代码，这意味着你可以很方便的在一个窗口内“一边看代码，一边看结果”。
  5. 5.易学易上手。Python这门语言的语法十分的接近自然英语，也尽量减少了不必要的符号的出现；同时，由于大量“包”的存在，所以对初学者非常友好。

当然，Python也并不是那么完美的，Python最受人诟病的一个特性在于，性能低。同样的一段程序，用C实现所需要的时间会远远小于python，所以在对计算性能要求较高的领域（如纯计算或者大型游戏），python的表现可能会不尽如人意。

不过，对于现代计算机的一般业务来说，即使是Python这样性能不高的编程语言，大多数情况下，实际运行过程中所花费的时间对人来说也并不是那么不可忍受。尤其是涉及网络通信和一般的文件读写时，花在通信和读写上的时间远远大于程序内部运算的时间，比如，通信和读写花了1秒，运算上两种语言分别花了0.001S和0.01S。即使有十倍的性能差距，但对人来说这1.001s和1.01秒就没什么差距了。同时，实在需要高性能的纯计算部分，python也可以调用你写好的高性能代码（如C）来保持性能。

看到这，你已经对这门语言有了一个初步的认识。不过，需要提醒大家的是，Python这门语言入门容易，但一些高级语言特性的掌握、一些复杂现实功能的实现仍然需要大家多费功夫。通常可以认为Python的学习曲线为先平后陡，所以大家也需要做好一定的心理准备。

当然，我们这篇教程只是入门教程，所以就无良地略过一些复杂、难懂的地方了。那么，让我们开始吧！
